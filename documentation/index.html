Getting Started

This guide will serve you as a guide for installing Erabo Pre-Selector

Minimum Requirements

In order to use the latest version of Pre-Selector, you must be running macOS High Sierra (10.13.4) or newer. You don’t need to have gigabytes of free space, and runs perfectly on any machine.

Download

You can download the lastest version on Pre-Selector in the App Store clicking here.

The Interface

Pre-Selector is minimal by design. Mantaining MacOS interface design, it layers the app in windows.

Importing Images

just drag and drop your camera images folder directly into Pre-Selector or clic the folder icon. Don't worry about having duplicates of RAW files and JPG files, Pre-Selector asosiate both files into one.

You can also go to File > Import Images or pressing ⇧ + ⌘ + I
