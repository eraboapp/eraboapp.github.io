Getting Started

Erabo Pre-Selector is an application that allows you to bulk-compress and apply a watermark to a large image set.

Whether you're a Photographer working on multiple wedding sessions or a Stock Image Website, you can benefit greatly from Erabo pre-selection flow. – Inspect, rate, and stamp your logo to hundreds of photos at lightning speed.

Minimum Requirements

In order to use the latest version of Pre-Selector, you must be running macOS High Sierra (10.13.4). You don’t need to have gigabytes of free space, and runs perfectly on any machine.
